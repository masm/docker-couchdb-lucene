# docker-couchdb-lucene

> A CouchDB Lucene base image built from `masm/archlinux`

## Install

Pull this image with `docker pull masm/couchdb-lucene`.

## Usage

In your Dockerfile, include the line `FROM masm/couchdb-lucene`.
